Station Data

The following datasets are available at the station level:

Format Link Description
JSON live.json Current live record. Updated every 48 or 300 seconds.
JSON samplerange.json Timestamps for the oldest and latest samples in the database.
JSON about.json Data for about page (navigation links, available stations, station and site name, etc). Also available at the same level as about.html.
JSON current_day_records.json Records for the current day.
JSON current_day_rainfall_totals.json Rainfall totals for the current day and past seven days.
JSON 24hr_rainfall_totals.json Rainfall totals for the last 24 and 168 hours (one day and seven days back from the current time).
JSON current_day_samples.json Samples for the current day.
JSON 24hr_samples.json Samples for the last 24 hours.
JSON current_day_7day_30m_avg_samples.json 30-minute averaged samples for the past seven days.
JSON 168hr_30m_avg_samples.json 30-minute averaged samples for the past 168 hours (like the 7-day data source but from the current time except current day).
JSON current_day_rainfall.json Rainfall for the current day.
JSON 24hr_hourly_rainfall.json Rainfall for the past 24 hours.
JSON current_day_7day_rainfall.json Rainfall for the past seven days.
JSON 168hr_rainfall.json Rainfall for the past 168 hours.
Google DataTable JSON datatable/current_day_samples.json Samples for the current day.
Google DataTable JSON datatable/24hr_samples.json Samples for the last 24 hours.
Google DataTable JSON datatable/current_day_7day_30m_avg_samples.json 30-minute averaged samples for the past seven days.
Google DataTable JSON 168hr_30m_avg_samples.json 30-minute averaged samples for the past 168 hours (like the 7-day data source but from the current time except current day).
Google DataTable JSON datatable/current_day_rainfall.json Rainfall for the current day.
Google DataTable JSON datatable/24hr_hourly_rainfall.json Rainfall for the past 24 hours.
Google DataTable JSON datatable/current_day_7day_rainfall.json Rainfall for the past seven days.
Google DataTable JSON datatable/168hr_rainfall.json Rainfall for the past 168 hours.
